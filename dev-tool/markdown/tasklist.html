                                让你的 VuePress 站点中的 Markdown 文件支持任务列表。

                                # 语法

                                • 使用 - [ ] 一些文字 渲染一个未勾选的任务项
                                • 使用 - [x] 一些文字 渲染一个勾选了的任务项 (我们也支持大写的 X)

                                # 例子

                                • [ ] Plan A
                                • [x] Plan B
                                - [ ] Plan A
- [x] Plan B
